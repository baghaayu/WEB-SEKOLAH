Teknik Komputer & Jaringan (TKJ)

Jurusan Unggulan dengan Sertifikasi MikroTik MTCNA

Deskripsi Jurusan

Jurusan TKJ mempelajari tentang instalasi, konfigurasi, dan troubleshooting jaringan komputer. Siswa akan dibekali dengan kemampuan untuk merancang, membangun, dan mengelola jaringan komputer baik skala kecil maupun besar. Program ini mencakup jaringan kabel, nirkabel, keamanan jaringan, dan administrasi server.

Kompetensi Lulusan

• Teknisi Jaringan Komputer
• Network Administrator
• System Administrator
• IT Support Specialist

Sertifikasi

• MikroTik MTCNA
• Cisco CCNA
• CompTIA Network+
• BNSP Teknisi Jaringan

Mitra Industri

• Telkom Indonesia
• Huawei
• Cisco Networking Academy
• Biznet Networks

Mata Pelajaran Unggulan

  • Jaringan Dasar: Konsep dasar jaringan komputer
  • Administrasi Server: Konfigurasi Windows & Linux Server
  • Keamanan Jaringan: Firewall, VPN, dan sistem keamanan
  • Pemrograman Web: HTML, CSS, JavaScript dasar
  • MikroTik MTCNA: Konfigurasi router MikroTik
  • Jaringan Nirkabel: WiFi, Access Point, Wireless Security

Fasilitas Praktik

• 3 Lab Komputer dengan spesifikasi tinggi
• Lab MikroTik khusus
• Lab Cisco Networking
• Server room dengan rack mount
• Peralatan jaringan lengkap (router, switch, access point)

Rekayasa Perangkat Lunak (RPL)

Jurusan Pengembangan Aplikasi dan Software

Deskripsi Jurusan

Jurusan RPL fokus pada pengembangan perangkat lunak termasuk pembuatan aplikasi web, mobile, dan desktop. Siswa akan belajar pemrograman, database management, UI/UX design, dan software engineering. Program ini dirancang untuk mencetak developer yang siap kerja di industri teknologi.

Kompetensi Lulusan

• Web Developer
• Mobile App Developer
• Software Engineer
• Database Administrator

Teknologi yang Dipelajari

• HTML/CSS/JavaScript
• PHP & Laravel
• Python & Django
• React Native

Prospek Kerja

• Startup Technology
• Software House
• Perusahaan IT
• Freelance Developer

Proyek yang Dikerjakan Siswa

  • Sistem Informasi Sekolah: Aplikasi web untuk manajemen sekolah
  • E-commerce Website: Toko online dengan payment gateway
  • Aplikasi Mobile: Android & iOS apps untuk berbagai kebutuhan
  • Game Development: Game 2D sederhana menggunakan Unity
  • IoT Projects: Sistem monitoring berbasis web

Otomatisasi Tata Kelola Perkantoran (OTKP)

Jurusan Administrasi Perkantoran Modern

Deskripsi Jurusan

Jurusan OTKP mempelajari manajemen perkantoran modern dengan penguasaan teknologi informasi, komunikasi bisnis, administrasi keuangan, dan manajemen dokumen. Program ini dirancang untuk mencetak tenaga administrasi profesional yang menguasai digital office.

Kompetensi Utama

• Administrasi Perkantoran
• Manajemen Dokumen
• Komunikasi Bisnis
• Keuangan Kantor

Software yang Dikuasai

• Microsoft Office Suite
• Google Workspace
• Accounting Software
• Digital Archiving System

Prospek Kerja

• Sekretaris Profesional
• Administrative Assistant
• Office Manager
• Data Entry Specialist

Praktik Simulasi Kantor

Fasilitas kantor simulasi lengkap dengan:

  • Reception Area: Simulasi menerima tamu dan telepon
  • Meeting Room: Praktek rapat dan presentasi
  • Filing System: Digital dan fisik document management
  • Accounting Section: Praktek pembukuan dan keuangan
